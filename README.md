# CREATIVE LOOP: Dinamiche di Replicazione Informazionale e Stabilità.
## Autore: Massimiliano Gasparini.
**Abstract**

Questo articolo formalizza il Model Autophagy Disorder (MAD) come un degrado dinamico della varianza informazionale nei sistemi di apprendimento generativo a ciclo chiuso. Superando le attuali metriche puramente empiriche, introduciamo il Creative Loop, un framework di replicazione non lineare che cattura matematicamente l'interazione tra informazione esterna (di origine umana) e informazione sintetica (generata dal modello). Il sistema viene modellato attraverso un set di equazioni differenziali accoppiate che esibiscono una soglia di transizione di fase tra un regime di stabilità e uno di collasso informazionale. 

/diagrams: Contiene i ritratti di fase, i diagrammi di biforcazione e la topologia del sistema aggiornati alla Versione 2.

/simulations: Contiene i log, i dataset o gli script (se presenti) utilizzati per validare le derivate parziali del modello.
